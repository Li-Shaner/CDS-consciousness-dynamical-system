## Abstract

This paper proposes a formal mathematical framework for consciousness grounded in a set of foundational postulates. Within this framework, consciousness is characterized as a formalizable dynamical system termed the Consciousness Dynamical System (CDS). Under a unified variable scheme, the system's core equation is derived:

C = fd / (a + l)

The framework aims to provide a formally derivable structure applicable to life–information systems, cognitive dynamics, and artificial system design.
